# GradePilot Installer Privacy Policy

Effective July 2026

## Introduction
This policy covers the standalone Windows Setup Wizard designed to install the GradePilot browser extension. The installer is built to configure your local environment safely and privately. 

## Data Access
The installer never reads your personal files. It does not transmit any data over the internet. It does not connect to any external servers during the installation process. 

The software strictly performs local system checks. It verifies your current Windows version and checks if Google Chrome is installed on your machine. 

## System Changes
The installer creates a dedicated folder in your local application data directory to hold the extension files. It creates standard desktop and start menu shortcuts for your convenience. It also registers the software with Windows so you can uninstall it cleanly at any time. 

## Telemetry
We do not include hidden tracking software inside the executable. The installation process is completely private and happens entirely on your machine.
